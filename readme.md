# TechGear Affiliate Website

## Overview
This is a modern tech affiliate website showcasing the best products for gaming, AI/ML, coding, peripherals, and networking. It is built using HTML, CSS, and JavaScript, with data dynamically loaded from a JSON file.

### Features
- Dark theme with a neon green accent inspired by Cash App
- Responsive design for desktop and mobile
- Card-based grid layouts with hover effects
- Dynamic product display using JavaScript and `fetch` API

### File Structure
- `index.html`: The main page of the site
- `css/style.css`: All styles and animations
- `js/app.js`: Dynamic JavaScript functionality
- `data/products.json`: Product data file

### Running the Site
Simply open `index.html` in any modern web browser.
