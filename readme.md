# KAEVA — The Best Tech, Curated for You

## Overview
Modern tech affiliate website showcasing the best products for gaming, AI/ML, coding, peripherals, and networking. Built with HTML, CSS, and JavaScript, powered by Vite.

### Features
- Dark theme with neon green accent (Cash App inspired)
- Responsive design for desktop and mobile
- Card-based grid layouts with hover effects
- Animated particle background, scroll reveals, gradient text
- Dynamic product display from JSON via `fetch` API

### File Structure
- `index.html` — Main page
- `css/style.css` — Styles and animations
- `js/app.js` — Dynamic functionality
- `data/products.json` — Product data
- `vite.config.js` — Build config

### Development
```bash
npm install
npm run dev      # Dev server on :3000
npm run build    # Production build → dist/
npm run preview  # Preview production build
```
